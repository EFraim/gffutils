gffutils.biopython_integration.to_seqfeature¶

gffutils.biopython_integration.to_seqfeature(feature)[source]¶

Converts a gffutils.Feature object to a Bio.SeqFeature object.

The GFF fields source, score, seqid, and frame are stored as qualifiers. GFF attributes are also stored as qualifiers.

Parameters:

feature : Feature object, or string

If string, assume it is a GFF or GTF-format line; otherwise just use the provided feature directly.
