gffutils.interface.FeatureDB.count_features_of_type¶

FeatureDB.count_features_of_type(featuretype=None)[source]¶

Simple count of features.

Can be faster than “grep”, and is faster than checking the length of results from gffutils.FeatureDB.features_of_type().

Parameters:

featuretype : string

Feature type (e.g., “gene”) to count. If None, then count all features in the database.
Returns:

The number of features of this type, as an integer
